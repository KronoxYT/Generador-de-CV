# **App Name**: VitaeForge

## Core Features:

- Profile Information Input: Allow users to input personal details, work experience, education, skills, and other relevant information to build their CV.
- Template Selection: Offer a variety of modern and professional CV templates to choose from.
- AI-Powered Content Refinement: Provide suggestions and improvements to the CV content, using an AI tool to optimize wording, grammar, and overall impact.
- Customization Options: Enable users to customize templates with different fonts, colors, and layouts.
- CV Preview: Display a real-time preview of the CV as it is being created and customized.
- PDF Export: Generate a downloadable PDF version of the completed CV.

## Style Guidelines:

- Primary color: Soft blue (#64B5F6) to convey trust and professionalism.
- Background color: Very light blue (#F0F8FF) to provide a clean and calming backdrop.
- Accent color: Muted grey (#90A4AE) for subtle highlights and divisions.
- Headline font: 'Poppins' sans-serif for a modern, precise feel.
- Body font: 'PT Sans' sans-serif which is paired well with 'Poppins' for longer pieces of text.
- Use a clean and structured layout to present information in an easy-to-read format.
- Incorporate professional and subtle icons to visually enhance sections and important information.
